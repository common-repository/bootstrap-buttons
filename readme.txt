== Description ==

A WordPress plugin that allows you to use Twitters Bootstrap buttons in your content.

The plugin offers 4 buttons sizes (Large, Small and Mini), 7 Button styles/colors (Default, Primary, Info, Success, Warning, Danger and Inverse) and 140 Icons in both Black and White versions.

Includes a shortcode inserter to make inserting your buttons hassle free in both the visual and html editor.

Built using [Caldera Engine](http://docs.caldera.co.za/get-caldera), the Pro version of [My Shortcodes](http://wordpress.org/plugins/my-shortcodes/) and exported as a WordPress Plugin.

== Installation ==

1. Upload the plugin folder 'bootstrap-buttons' to your `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A Bootstrap Buttons button will be available above the Content Editor.
